An open source implementation of Neural Voice Cloning with Few Samples


  • Github: https://github.com/r9y9/deepvoice3_pytorch

This page provides audio samples from the speaker adaptation approach of the open source implementations Neural Voice Cloning with Few Samples. Samples from single speaker and multi-speaker models follow.

The model is first trained on 84 speakers. Then the model is adapted to a particular speaker to generate clone samples
